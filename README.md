# GoGo 

**GoGo** is a simple and fun voice-controlled jumping game made with Python and Pygame!  
Jump over obstacles by making sounds (how loud you are decides how high you jump) get your frustrations out (and you can just tell people you were playing this fun new game!).

## Features:
- Voice-controlled jumps using your microphone 
- Increasing score for each obstacle you pass 
- Game over screen with restart option (press **R**) 
- Cute and simple design with a moving blob and obstacles 

## How to Play:
1. Run `main.py` (make sure you have Python, Pygame, and SoundDevice installed)
2. Make a sound to jump over obstacles
3. Avoid hitting the red rectangles
4. Score points for each obstacle you pass
5. Press **R** to restart if you lose
